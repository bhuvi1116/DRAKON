# Быстрое соединение узлов на canvas v0

На этом шаге в редактор добавлен быстрый canvas-first режим соединения узлов.

## Что добавлено

- временная линия предпросмотра от узла-источника к текущей позиции указателя;
- подсветка узла-источника, выбранного для создания связи;
- завершение связи простым левым кликом по целевому узлу.

## Базовый сценарий

1. Выбрать узел как источник связи через контекстное меню.
2. Переместить указатель по canvas и увидеть временную линию.
3. Щёлкнуть по целевому узлу.
4. Связь создаётся и попадает в историю Undo/Redo.

## Ограничения

- пока не реализован drag-wire от специальных портов;
- условные порты выбираются по текущим базовым правилам редактора;
- отмена режима соединения по клавише Escape будет добавлена позднее.
