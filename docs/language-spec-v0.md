# Спецификация языка v0

## Узлы
- Start
- Action
- Condition
- End

## Ограничения
- Ровно один Start
- Минимум один End
- Start не имеет входов
- End не имеет выходов
- Condition должен иметь ровно две исходящие ветви: True и False
- На этапе v0 циклы запрещены

## Семантика
- Start: вход в алгоритм
- Action: простое действие, например `x = 1` или `print(x)`
- Condition: булево выражение
- End: завершение алгоритма
