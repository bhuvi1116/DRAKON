# Работа редактора с файлами диаграмм v0

На этом этапе редактор умеет работать с файлами `.drakon.json` как с обычными проектами.

## Поддерживаемые операции

- открыть существующую диаграмму по пути;
- сохранить текущую диаграмму;
- сохранить диаграмму под новым именем;
- автоматически синхронизировать путь текущего проекта в панели состояния.

## Ограничения этапа

- выбор файла пока выполняется через текстовое поле пути, без нативного file picker;
- состояние `dirty` подготовлено как инфраструктура, но полноценно раскроется после добавления интерактивного редактирования диаграммы.

## Следующий шаг

Следующим шагом должна стать история изменений: `Undo / Redo` для действий редактора и обновление состояния документа после реальных операций редактирования.
