# Drag-and-drop foundation v0

На этом шаге добавлена прикладная основа для drag-and-drop перемещения узлов на canvas.

Что сделано:
- введен `NodeDragService`;
- выделена модель drag-сессии `DragSession`;
- описан перенос координат по дельте указателя;
- сохранена совместимость с уже введенной привязкой к сетке.

Это не финальный visual canvas, а подготовительный слой, который позволит на следующем шаге связать pointer events Avalonia с реальным перемещением узлов в редакторе.
