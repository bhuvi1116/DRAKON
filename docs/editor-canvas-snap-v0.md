# Canvas Snap-to-Grid v0

На шаге 19 drag-and-drop узлов на canvas завершает перемещение автоматической привязкой к сетке.

Что реализовано:
- итоговая позиция узла округляется к ближайшей точке сетки;
- результат drag сохраняется в command-based Undo/Redo истории;
- статус canvas и история изменений явно сообщают о привязке.

Текущее поведение:
- во время перетаскивания узел движется свободно;
- после отпускания указателя позиция нормализуется по текущему шагу сетки;
- при размере сетки <= 0 используется fallback-значение из ViewModel.

Следующий этап:
- визуально показывать snap-target во время drag;
- подсвечивать активный узел и drag-state;
- подготовить контекстные операции на canvas.
