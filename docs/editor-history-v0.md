# История изменений редактора v0

На шаге 11 в редактор добавлена базовая snapshot-история документа.

Что входит:
- Undo
- Redo
- фиксация снимка перед изменением документа
- сброс истории при открытии файла или создании проекта из шаблона

На текущем этапе история покрывает:
- переименование документа
- замещение текущего документа при открытии файла
- замещение текущего документа при создании проекта из шаблона

Ограничения:
- история пока snapshot-based, а не command-based;
- операции canvas-редактирования будут подключаться на следующем шаге;
- история не сериализуется в файл проекта.
