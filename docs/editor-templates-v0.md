# Шаг 9: шаблоны проектов в редакторе

На этом шаге в Avalonia-редактор добавлен сценарий создания нового проекта прямо из UI.

## Что умеет редактор

- выбрать шаблон из каталога:
  - `hello-world`
  - `minimal`
  - `simple-branch`
  - `max-of-two`
- задать имя проекта;
- указать путь сохранения `.drakon.json`;
- создать файл проекта без участия CLI;
- сразу загрузить созданную диаграмму в текущий рабочий контекст редактора;
- после этого продолжить export/build/run pipeline.

## Архитектура

Добавлен сервис `TemplateBootstrapService`.

Он отвечает за:
- получение списка шаблонов;
- создание документа из шаблона;
- переименование проекта;
- сохранение `.drakon.json`;
- возврат результата для загрузки в `MainWindowViewModel`.

Такое разделение позволяет тестировать bootstrap отдельно от Avalonia UI.

## Ограничения текущего шага

- выбор пути пока текстовый, без native file picker;
- нет диалога подтверждения перезаписи файла;
- шаблоны пока создаются только из встроенного каталога.
