# Редактор: координаты узлов и привязка к сетке v0

На этом этапе редактор получил базовое управление положением узлов.

Что доступно:
- просмотр координат выбранного узла;
- ручное изменение X/Y;
- привязка выбранного узла к сетке с заданным шагом;
- Undo/Redo для изменения положения.

Ограничения текущей версии:
- перемещение пока выполняется через панель свойств, а не drag-and-drop на canvas;
- сетка логическая, без визуальной отрисовки;
- массовое выравнивание и snap-to-grid для нескольких узлов пока отсутствуют.

Следующее развитие:
- drag-and-drop перемещение;
- выделение на canvas;
- визуальная сетка;
- выравнивание группы узлов.
