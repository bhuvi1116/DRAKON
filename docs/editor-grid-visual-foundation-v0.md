# Визуальная основа сетки редактора

## Цель

На этом шаге в проект добавлена инженерная основа для визуальной сетки canvas. Сетка пока еще не подключена к реальному Avalonia drawing pipeline, но логика расчета линий вынесена в отдельный сервис и покрыта unit tests.

## Что добавлено

- `GridRenderService`
- модель `GridLine`
- перечисление `GridLineOrientation`
- unit tests для расчета вертикальных и горизонтальных линий

## Почему это выделено отдельно

Сетка относится к визуальному слою, но ее расчет должен быть:
- детерминированным;
- повторно используемым;
- независимым от конкретного canvas-контрола;
- пригодным для будущего zoom/pan.

Поэтому расчет линий сетки вынесен в сервис, а не зашит напрямую в code-behind окна.

## Следующий шаг

Следующим шагом этот сервис подключается к реальному canvas Avalonia:
- отрисовка линий сетки;
- визуальное выделение узлов;
- pointer-driven drag-and-drop;
- snap-to-grid уже в визуальном режиме.
