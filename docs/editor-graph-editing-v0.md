# Редактирование графа в редакторе v0

На этом шаге редактор получил минимальные операции изменения диаграммы прямо из UI:

- добавление узлов `Start`, `Action`, `Condition`, `End`;
- удаление выбранного узла;
- редактирование текста выбранного узла;
- создание связи между двумя выбранными узлами;
- удаление выбранной связи;
- подключение этих операций к snapshot-истории `Undo/Redo`.

Пока это не полноценный canvas-редактор с drag-and-drop, но это уже рабочий управляемый контур изменения графовой модели без ручного редактирования JSON.

## Ограничения текущей версии

- выбор выполняется через списки узлов и связей;
- нет визуального соединения мышью на canvas;
- история пока snapshot-based;
- нет перемещения узлов мышью и нет undo по координатам.

## Следующий шаг

Следующий этап должен перевести snapshot-подход к более явной command-based модели и добавить операции перемещения, свойства узлов и более наглядный canvas workflow.
