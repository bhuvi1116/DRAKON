# Шаблоны проектов и версионирование формата

На этом шаге проект получает два практических механизма:

- version-aware сериализацию диаграмм;
- CLI-команду для создания новых файлов из шаблонов.

## Версия формата

Поле `version` хранится в JSON каждой диаграммы.

Текущая версия формата:
- `0.1`

Если редактор или CLI открывают диаграмму с другой версией, валидатор выдает предупреждение совместимости. Это не блокирует чтение, но явно сигнализирует, что проект был создан другой версией формата.

## Шаблоны

CLI теперь умеет создавать стартовые файлы диаграмм без ручного копирования sample-каталогов.

Поддерживаемые шаблоны:
- `hello-world`
- `minimal`
- `simple-branch`
- `max-of-two`

Пример:

`dotnet run --project src/Cli/Cli.csproj -- new hello-world ./work/hello.drakon.json`

Команда:
- создает каталог, если его еще нет;
- сериализует шаблон в JSON;
- печатает путь к созданному файлу;
- возвращает ненулевой код выхода при неизвестном имени шаблона.
