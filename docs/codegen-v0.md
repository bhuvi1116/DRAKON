# Кодогенерация v0

## Цель

На этом шаге диаграмма преобразуется в минимальное структурное IR, после чего из него печатается C99-код.

## Поддерживаемые конструкции

- линейная последовательность `Start -> Action -> ... -> End`
- условие `Condition` с двумя ветвями `True/False`
- простые действия:
  - `x = 1`
  - `result = a + b`
  - `print(result)`
  - `print("text")`

## Ограничения

- циклы запрещены и отсекаются валидатором
- кодогенерация ориентирована на структурные ациклические диаграммы
- тип переменных пока фиксирован как `int`
- отсутствует отдельный runtime

## Пайплайн

```text
DiagramDocument
  -> DiagramValidator
  -> NormalizedFlowBuilder
  -> FlowNode IR
  -> CPrinter
  -> main.c
```
