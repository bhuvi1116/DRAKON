# Editor Canvas Connection Cancel v0

## Что добавлено

На этом этапе canvas-first режим создания связи стал более управляемым:

- режим создания связи можно отменить клавишей `Escape`;
- режим создания связи можно отменить кликом в пустую область canvas;
- при наведении на допустимую цель соединения узел получает визуальную подсветку;
- редактор явно различает недопустимые и допустимые цели для завершения связи.

## Поведение

### Отмена по Escape

Если пользователь уже выбрал источник связи, нажатие `Escape`:
- очищает pending-source состояние;
- убирает временную линию предпросмотра;
- возвращает canvas в обычный режим.

### Отмена по клику в пустую область

Если режим создания связи активен и пользователь кликает в пустую область:
- pending-source очищается;
- preview-связь исчезает;
- в `CanvasSummary` записывается причина отмены.

### Подсветка допустимой цели

Во время перемещения указателя по canvas:
- если под указателем есть узел, который допустим как цель связи,
  он подсвечивается как возможная цель;
- если узел недопустим, специальная подсветка не показывается.

## Зачем это нужно

Это делает режим соединения узлов менее хрупким и ближе к ожидаемому UX редакторов диаграмм:
- у пользователя есть явный выход из режима;
- допустимые цели становятся заметнее;
- уменьшается вероятность случайного создания связи.
