# Canvas context operations v0

На этом шаге в `DiagramCanvasControl` добавлены базовые контекстные операции прямо на диаграмме.

## Что доступно

### Для узла
- использовать узел как источник связи;
- быстро завершить связь на выбранный узел, если источник уже установлен;
- удалить выбранный узел.

### Для связи
- удалить выбранную связь.

## Поведение

- правый клик по узлу открывает меню операций узла;
- правый клик по линии связи открывает меню операций связи;
- выбор синхронизируется с `MainWindowViewModel`;
- быстрые операции используют уже существующие сервисы редактирования и историю Undo/Redo.

## Ограничения v0

- пока нет визуального ghost-wire предпросмотра новой связи;
- меню создается программно внутри canvas-контрола;
- hit testing связи основан на расстоянии до отрезка между центрами узлов.
