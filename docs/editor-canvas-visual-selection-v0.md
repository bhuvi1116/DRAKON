# Editor Canvas Visual Selection v0

На этом шаге редактор получил реальный кастомный canvas-контрол Avalonia.

Что добавлено:
- отрисовка сетки по текущему шагу GridSize;
- отрисовка узлов и связей;
- визуальное выделение выбранного узла;
- синхронизация выбора canvas ↔ свойства узла ↔ списки редактора.

Ограничения:
- drag-and-drop еще не подключен;
- routing связей пока прямолинейный;
- нет zoom/pan.
