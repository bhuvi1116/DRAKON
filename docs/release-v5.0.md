# DRAKON Studio 5.0

## Что входит в релиз

- visual-first редактор диаграмм ДРАКОН
- шаблоны и bootstrap новых проектов
- JSON сериализация и файловый workflow
- command-based Undo/Redo
- canvas selection, drag-and-drop и quick-connect
- visual ports и силуэтный layout
- DRAKON Auto-layout v5.0
- C code generation для исполнимого профиля
- экспорт generated проекта и сборка через CMake

## Ограничения

- исполнимый pipeline пока охватывает только подмножество узлов
- для visual-профиля кодогенерация преднамеренно ограничена
- для точной сверки с PDF-спецификацией файл спецификации нужно загрузить заново, так как предыдущая копия больше недоступна в среде
